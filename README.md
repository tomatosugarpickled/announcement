# GlobalGates 담당 기능 소개

GlobalGates 프로젝트에서 제가 맡은 부분은 사용자가 서비스를 처음 시작하고, 자신의 정보를 관리하는 흐름입니다.

크게 세 가지 기능을 담당했습니다.

- 회원가입 / 로그인
- 마이페이지
- 설정

AI 기능은 별도 repository에서 정리할 예정이라 이 문서에서는 제외했습니다.

---

## 1. 회원가입 / 로그인

사용자가 GlobalGates를 처음 이용하기 위해 필요한 가입과 로그인 기능을 만들었습니다.

회원가입 과정에서는 사용자가 입력한 정보가 이미 사용 중인지 확인하고, 전화번호나 이메일 인증을 통해 실제 본인이 사용하는 연락처인지 확인할 수 있도록 했습니다.

로그인 후에는 사용자가 마이페이지와 설정 페이지 같은 개인 기능을 사용할 수 있습니다.

### 주요 기능

- 회원가입
- 로그인
- 이메일 중복 확인
- 전화번호 중복 확인
- 사용자 아이디 중복 확인
- 회사명 / 사업자번호 중복 확인
- SMS 인증
- 이메일 인증
- 프로필 이미지 등록
- SNS 로그인 후 추가 정보 입력
- 비활성 계정 다시 사용하기

### 사용자가 경험하는 흐름

1. 회원가입 화면에 들어갑니다.
2. 이메일, 전화번호, 사용자 아이디 등을 입력합니다.
3. 이미 사용 중인 정보인지 확인합니다.
4. 전화번호 또는 이메일 인증을 진행합니다.
5. 프로필 이미지를 등록할 수 있습니다.
6. 가입이 완료되면 로그인할 수 있습니다.
7. 로그인 후 마이페이지와 설정 페이지를 사용할 수 있습니다.

### JWT 로그인 방식

로그인 후에는 JWT라는 토큰을 사용했습니다.

JWT는 사용자가 로그인했다는 것을 확인해주는 디지털 출입증 같은 역할을 합니다. 로그인에 성공하면 서버가 토큰을 발급하고, 이후 마이페이지나 설정 페이지에 들어갈 때 이 토큰을 통해 로그인한 사용자인지 확인합니다.

토큰 안의 정보는 암호화된 형태로 다뤄지기 때문에, 사용자의 로그인 상태를 더 안전하게 확인할 수 있습니다. 또한 다른 사람이 임의로 사용자인 척 요청하는 것을 막는 데 도움이 됩니다.

### SNS 로그인

SNS로 로그인한 사용자는 바로 가입이 끝나는 것이 아니라, GlobalGates에서 필요한 추가 정보를 입력해야 합니다.

SNS에서는 이름이나 이메일 정도만 받아올 수 있지만, GlobalGates는 사업자 정보, 사용자 아이디, 관심 분야 같은 정보가 필요하기 때문입니다.

그래서 SNS 로그인 후에도 필요한 정보를 한 번 더 입력하도록 구성했습니다.

## 2. 마이페이지

마이페이지는 사용자가 자신의 프로필과 활동 내역을 확인하는 공간입니다.

내 마이페이지에서는 프로필을 수정하거나 내가 등록한 상품과 게시글을 볼 수 있습니다. 다른 사람의 마이페이지에 들어가면 팔로우, 차단, 신고, 채팅 같은 기능을 사용할 수 있습니다.

### 주요 기능

- 내 프로필 보기
- 다른 사람 프로필 보기
- 프로필 이미지 변경
- 배너 이미지 변경
- 내 상품 목록 보기
- 내 게시글 보기
- 내가 쓴 댓글 보기
- 좋아요한 게시글 보기
- 견적 요청 내역 보기
- 상품 등록
- 상품 삭제
- 팔로우 / 언팔로우
- 차단
- 신고
- 채팅으로 이동

### 사용자가 경험하는 흐름

1. 로그인 후 마이페이지에 들어갑니다.
2. 내 프로필과 활동 내역을 확인합니다.
3. 상품, 게시글, 댓글, 좋아요 탭을 이동하면서 내 활동을 볼 수 있습니다.
4. 프로필 수정 버튼을 눌러 닉네임, 소개글, 프로필 이미지, 배너 이미지를 바꿀 수 있습니다.
5. 상품을 등록하거나 삭제할 수 있습니다.
6. 다른 사람의 마이페이지에서는 팔로우, 차단, 신고, 채팅 기능을 사용할 수 있습니다.

### 신경 쓴 부분

마이페이지는 내 페이지와 다른 사람의 페이지가 다르게 보여야 했습니다.

내 페이지에서는 수정과 관리 기능이 필요하고, 다른 사람의 페이지에서는 팔로우나 차단 같은 기능이 필요합니다. 그래서 같은 마이페이지 화면 안에서도 사용자가 누구인지에 따라 보이는 버튼과 기능이 달라지도록 만들었습니다.

또한 상품, 게시글, 댓글, 좋아요처럼 볼 수 있는 내용이 많아서 탭으로 나누어 정리했습니다. 사용자가 원하는 활동 내역만 쉽게 확인할 수 있도록 구성했습니다.

## 3. 설정

설정 페이지는 사용자가 가입 후에도 자신의 계정 정보를 직접 관리할 수 있는 공간입니다.

전화번호, 이메일, 사용자 아이디 같은 기본 정보뿐만 아니라 알림, 언어, 차단 목록 같은 사용 환경도 관리할 수 있습니다.

### 주요 기능

- 사용자 아이디 변경
- 전화번호 변경
- 이메일 변경
- 비밀번호 변경
- 계정 비활성화
- 알림 켜기 / 끄기
- 세부 알림 설정
- 언어 변경
- 국가 변경
- 차단한 사용자 목록 보기
- 차단 해제
- 구독 상태 확인
- 구독 해지

### 사용자가 경험하는 흐름

1. 설정 페이지에 들어갑니다.
2. 계정 정보를 확인합니다.
3. 사용자 아이디, 전화번호, 이메일을 변경할 수 있습니다.
4. 비밀번호를 변경할 수 있습니다.
5. 알림을 켜거나 끌 수 있습니다.
6. 원하는 알림만 받을 수 있도록 세부 설정을 바꿀 수 있습니다.
7. 차단한 사용자 목록을 확인하고 차단을 해제할 수 있습니다.

### 신경 쓴 부분

설정 페이지는 사용자의 개인정보와 관련된 기능이 많기 때문에, 아무나 정보를 바꿀 수 없도록 처리했습니다.

예를 들어 사용자가 이메일이나 전화번호를 바꾸려면 중복 확인과 인증 과정을 거치도록 했습니다. 비밀번호 변경이나 계정 비활성화처럼 중요한 기능은 현재 비밀번호를 확인한 뒤 진행되도록 했습니다.

설정 기능은 메뉴가 많기 때문에 발표에서는 전부 보여주지 않고 핵심 기능만 보여줄 예정입니다.

### 발표에서 보여줄 설정 기능

- 사용자 아이디 변경
- 알림 설정
- 차단 목록 확인

전화번호 변경, 이메일 변경, 비밀번호 변경, 계정 비활성화는 실제 계정 상태가 바뀌거나 인증 과정이 필요하기 때문에 발표에서는 설명으로만 다룰 예정입니다.

## 발표 시연 흐름

발표에서는 아래 순서로 시연할 예정입니다.

1. 로그인
2. 마이페이지 이동
3. 프로필 수정 화면 확인
4. 상품 / 게시글 / 댓글 / 좋아요 탭 확인
5. 다른 사람 프로필에서 팔로우 또는 더보기 메뉴 확인
6. 설정 페이지 이동
7. 사용자 아이디 변경 흐름 확인
8. 알림 설정 확인
9. 차단 목록 확인
10. Swagger 이미지로 기능별 API 확인

## 발표 중 직접 실행하지 않을 기능

아래 기능은 구현되어 있지만, 발표 중에는 직접 실행하지 않을 예정입니다.

외부 인증이나 계정 상태 변경이 포함되어 있어서 시연 중 문제가 생길 수 있기 때문입니다.

- SMS 인증 전체 진행
- 이메일 인증 전체 진행
- SNS 신규 가입 전체 진행
- 비밀번호 변경
- 계정 비활성화
- 구독 해지

---

## 정리

제가 맡은 기능은 사용자가 GlobalGates를 시작하고, 자신의 정보를 관리하는 흐름입니다.

회원가입과 로그인에서는 사용자가 안전하게 서비스를 시작할 수 있도록 했습니다.

마이페이지에서는 자신의 프로필과 활동 내역을 확인하고 관리할 수 있도록 했습니다.

설정에서는 가입 이후에도 사용자가 계정 정보와 알림, 차단 목록 등을 직접 관리할 수 있도록 했습니다.
